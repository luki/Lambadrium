# Lambdarium
A simple implementation of a blockchain in Haskell.
_Work in proccess_

## Capabilities
### Current
* Creating a genesis block
* Adding blocks to the blockchain
* Hashing blocks
* Mining blocks (with set difficulty)

### Upcoming
* Verifying the blockchain
* (Rewarding a miner)
